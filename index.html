Projects

  • Retro Analytics

    Scrapes historical prices of old video games on ebay and posts/comments from Reddit. With this data, we train an ARIMA model on the historical data to see which games are likely to increase in value. We also pair these forecasts with general sentiment of the game in online communities.

  • Chinese Tutor

    I developed a personal Chinese tutor that is trained on my Chinese textbook. I am learning 中文 as my girlfriend is Taiwanese, so I wanted a tool that could help me learn along the way. The following bullet points are highlights of the project:

    • Scan the pdfs via an OCR scanner in Python and use GPT4 to correct errors in the scanning process in order to acquire high quality source data for the local LLM.

    • Store processed data on a vector db (pinecone) to be used for Retrieval Augmentation Generation (RAG).

    • Using the vector DB, I used a local LLM (llama2 7b from HF) to access the information from pinecone. This allowed the local llm to answer specific nuanced questions regarding my textbook.

    The primary advantage over using GPT4 is that, once the text extraction is complete, the model is entirely localized. I do not need to ping an API for requests, all I need is my eGPU. However, one limitation is that I'm using a 3070 ti, which only has 8 GB of VRAM. With this GPU, I can only feasibly run a 7b parameter model. Anything larger presents issues. My ideal setup would be two 4090s, though currently this is out of my budget range.
