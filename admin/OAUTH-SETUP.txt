Decap CMS OAuth via Cloudflare (decap-proxy)
=============================================

1. Deploy proxy:
     git clone https://github.com/sterlingwes/decap-proxy.git
     cd decap-proxy
     cp wrangler.toml.sample wrangler.toml   # set unique `name`
     npx wrangler login
     npx wrangler secret put GITHUB_OAUTH_ID
     npx wrangler secret put GITHUB_OAUTH_SECRET
     npx wrangler deploy
   Note the worker URL — your PROXY URL.

2. GitHub OAuth App: https://github.com/settings/applications/new
     Homepage URL  = PROXY URL
     Callback URL  = PROXY URL + /callback
   Put Client ID/Secret into the wrangler secrets above; redeploy.

3. Private repo only: set GITHUB_REPO_PRIVATE = "1" in wrangler.toml; redeploy.

4. admin/config.yml:
     base_url: PROXY URL
     auth_endpoint: auth

5. Test: /admin/ → Login with GitHub.

Refs:
- https://github.com/sterlingwes/decap-proxy
- https://decapcms.org/docs/github-backend/
